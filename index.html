Movie Sentiment Analysis

Developed a sentiment analysis model to classify movie reviews as positive or negative using a Simple Recurrent Neural Network (RNN). Utilized the IMDB dataset, preprocessed text data by tokenizing and padding sequences to a uniform length of 500 words. The model architecture included an embedding layer to convert words into dense vectors, followed by a Simple RNN layer to capture sequential patterns, and a dense output layer with a sigmoid activation function for binary classification. Achieved a test accuracy of 75%, demonstrating effective sentiment classification. This project showcases the application of RNNs in natural language processing tasks.
